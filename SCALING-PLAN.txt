OVERVIEW
---------
Turn the work done on Elevated Apartment Locating into
a repeatable service business. Build once, deploy fast,
own the infrastructure, collect monthly retainers.

The model: you own all the accounts. Clients pay a
setup fee + monthly retainer. They get a login and
a one-step DNS instruction. Nothing else required
from them.


-----------------------------------------------------
INFRASTRUCTURE (YOUR ACCOUNTS — ONE TIME SETUP)
-----------------------------------------------------

VERCEL PRO — $20/mo
  - Hosts all client sites under one account
  - Unlimited projects, custom domains per client
  - Auto-deploys when code is updated
  - Sign up: vercel.com

GITHUB (FREE or Team)
  - One organization, one private repo per client
  - Code lives here, Vercel pulls from it
  - Clients never touch this

RESEND (FREE to start)
  - Handles contact form emails for all clients
  - Configure each client's "from" address separately
  - Free tier: 3,000 emails/mo (upgrade if needed)
  - Sign up: resend.com

SANITY (FREE per project)
  - One organization, one project per client
  - You create a login for each client
  - Clients use this to manage reviews, content, etc.
  - Sign up: sanity.io

YOUR COSTS AT SCALE
  1-5 clients:    ~$20/mo (Vercel Pro)
  6-20 clients:   ~$30/mo (Vercel Pro + Resend upgrade)
  20+ clients:    ~$50/mo (scale Resend as needed)
  Margin is strong at any size.

-----------------------------------------------------
CLIENT EXPERIENCE (WHAT THEY ACTUALLY DO)
-----------------------------------------------------
Clients do exactly two things:

  1. SIGN + PAY
     Contract and setup fee. Everything else is on you.

  2. ADD ONE DNS RECORD
     You send them a single screenshot specific to their
     registrar (GoDaddy, Squarespace, etc.) with an arrow
     pointing to exactly where to click and what to paste.
     Most people can do this in under 5 minutes.

  3. RECEIVE LOGIN CREDENTIALS
     You email them their Sanity dashboard login.
     They can add/edit reviews, testimonials, etc.
     No technical knowledge needed.

That's it. They never hear about GitHub, Vercel,
Resend, API keys, or any of it.

THE TEMPLATE SYSTEM
-----------------------------------------------------
The apartment locating site (Elevated) is Template 01.
It has a clear structure that works for any similar
service business:

  - Hero section (photo + headline + CTA)
  - Gallery / proof section
  - How It Works (3 steps)
  - Our Story / About
  - Service areas or neighborhoods
  - Reviews (auto-scrolling, Google-sourced)

TO DEPLOY A NEW CLIENT:
  1. Fork the template repo
  2. Update: business name, city, colors, copy, photos
  3. Wire up their Resend email
  4. Add their Sanity project
  5. Deploy to Vercel under their domain
  Realistic time: 3-6 hours for a clean deployment

-----------------------------------------------------
REVIEW AUTOMATION PIPELINE
-----------------------------------------------------
Built into the Elevated template — replicate for
every new client:

  1. Client has Google Business Profile
  2. Nightly cron job pulls 4-5 star reviews via
     Google Places API
  3. Reviews land in Sanity as "pending"
  4. Client gets email: "You have new reviews to approve"
  5. Client logs into Sanity, approves with one click
  6. Site updates instantly

What client needs to provide (you handle the rest):
  - Google Business Place ID
  - Google Cloud API Key
  Both are free and take ~15 minutes to set up.
  Include instructions in your client onboarding doc.
